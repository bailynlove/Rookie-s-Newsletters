# Rookie's News — Domain Glossary

## Project

**Rookie's News** is a dual-stream intelligence dashboard hosted on GitHub Pages, fed by two autonomous agents:

- **Heimdall** — Academic intelligence (AI/ML papers, GitHub trending, HackerNews, arXiv subscriptions)
- **Friday** — Macro-finance intelligence (A-share, HK, US markets, L1-L4 macro factors)

## Terms

| Term | Definition |
|------|------------|
| **日报 (Daily Report)** | A single HTML file generated by an agent, covering a specific time period (morning/noon/evening for academic; morning/a-close/evening/us-premarket for finance). |
| **学术分区 (Academic Section)** | The `academic/` directory containing Heimdall's HTML outputs. |
| **宏观金融分区 (Macro-Finance Section)** | The `macro-finance/` directory containing Friday's HTML outputs. |
| **P0 / P1 / P2** | Priority tiers in Heimdall's reports. P0 = immediate attention (5★), P1 = worth deep dive (4★), P2 = nice to know (3★). |
| **L1-L4 框架** | Friday's macro factor hierarchy: L1 Global → L2 Transmission → L3 Domestic → L4 Market. |
| **订阅命中 (Subscription Hit)** | When an arXiv paper matches one of Heimdall's topic subscriptions (e.g. `self-improving-agent`, `mcp-tools`). |
| **趋势信号 (Trend Signal)** | A detected pattern across multiple data points, tagged with topic, strength (1-5), and direction (rising/stable/falling). |
| **情绪评分 (Sentiment Score)** | Friday's market sentiment indicator, 0-100 scale. |
| **置信度 (Confidence)** | Friday's assessment certainty for a market direction call, 0.0-1.0 scale. |
| **持仓标的 (Portfolio Holding)** | A specific stock or ETF that Friday tracks for the user. **Privacy-sensitive — must be filtered from public HTML.** |
| **观望标的 (Watchlist)** | A stock or ETF under observation but not held. **Privacy-sensitive — must be filtered from public HTML.** |

## File Conventions

- Agent configs live in `configs/<agent>/`
- HTML outputs live in `<section>/YYYY-MM-DD/<period>.html`
- Trend data lives in `data/*.json` (generated by `scripts/build-dashboard.py`)
- Dashboard homepage is `index.html`
- Styles are in `assets/css/main.css`

## Privacy Rules

Friday's HTML outputs **must not** contain:
- Portfolio holdings (持仓标的评估 sections)
- Watchlist items (观望标的评估 sections)
- Personal trading recommendations (操作建议 table rows with specific tickers)

These are stripped at generation time by the agent per `configs/friday/html-output-instructions.md`.
