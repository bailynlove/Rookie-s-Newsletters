MOSS: 自演化代理系统的源级重写框架

来源: arXiv:2605.22794

领域: cs.AI, cs.LG

作者: Qianshu Cai, Yonggang Zhang, Yike Guo 等（香港科技大学团队）

重要性: 首次提出 Agent 通过源级代码重写实现自演化。MOSS 框架让 Agent 能够自我修改、自我优化，突破传统 Agent 的固定能力边界。这是 Agent 自进化研究的重大突破，可能改变整个 Agentic 系统的架构范式。

评分: ★★★★★ (9.5/10)

建议路由: → Jarvis（技术架构评估）→ Pepper（论文深度解读）

Gated DeltaNet-2: 解耦线性注意力中的擦除与写入

来源: arXiv:2605.22791

领域: cs.AI

作者: Ali Hatamizadeh, Yejin Choi, Jan Kautz

重要性: Yejin Choi 与 Jan Kautz 团队推出 DeltaNet-2 技术报告。核心创新：解耦线性注意力机制的擦除与写入操作，解决传统线性注意力在长序列处理中的信息丢失问题。对 Transformer 替代架构研究有重要意义，可能影响下一代 LLM 架构设计。

评分: ★★★★★ (9.0/10)

建议路由: → Jarvis（架构分析）
