PACT: 协议化 Agent 通信框架 — 成本砍半、性能不变

来源: arXiv:2606.05304 (cs.AI)

重要性: 提出 PACT（Protocolized Action-state Communication and Transmission），将 Agent 间通信从自由文本转为精炼的 action-state 记录。在 SWE-agent 上输入 token 减半且 resolve 率不变，在 OpenHands 上 resolve 率提升同时 token 消耗降低 10%。对 STARK_TOWER 多 Agent 架构的直接优化参考。

评分: ★★★★★ (8.7/10)

模型塌缩流行病学模型 — 合成数据污染阈值预警

来源: arXiv:2606.05168 (cs.CL)

重要性: 首次将模型塌缩建模为跨层 SIR/SIRS 传染病动力学。推导出基本再生数 R₀ > 1 意味着合成数据污染将不可逆蔓延。1,088 轮 GPT-2 实验验证剂量-响应退化。关键发现：检测过滤与群体免疫是最高杠杆干预策略。对依赖合成数据 workflow 的所有团队构成架构警示。

评分: ★★★★★ (8.6/10)
